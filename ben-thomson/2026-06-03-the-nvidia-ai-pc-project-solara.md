# The Nvidia AI PC, Project Solara, Microsoft AI

**Date:** June 3, 2026  
**Source:** [Stratechery by Ben Thompson](https://stratechery.com/2026/the-nvidia-ai-pc-project-solara-microsoft-ai/)

Analysis of Nvidia's RTX Spark chip, Microsoft's Project Solara platform, and new homegrown MAI AI models.

### The Nvidia AI PC (RTX Spark)

Nvidia CEO Jensen Huang unveiled the RTX Spark (N1X) at Computex – a PC processor made alongside Microsoft for Windows laptops. Specs include up to 20 Arm CPU cores, Blackwell GPU (6,144 CUDA cores), 128GB LPDDR5X RAM.

Thompson's critique: "The ideal setup for a local agent is strong local CPU performance and calling out to the cloud for inference. The RTX Spark, however, spends tons of die space on GPU cores that are inferior to the cloud… at the expense of CPU."

### Project Solara

Microsoft's platform for devices that run AI agents instead of apps, built on Android (not Windows). Vision: "The next computer is not one device. It is all these devices working together as one system, with agents showing up closer to where and when you need them."

### Microsoft AI (MAI Models)

Microsoft unveiled seven homegrown AI models built from scratch. Flagship model: MAI-Thinking-1 – a reasoning model that draws even with Anthropic Claude Sonnet 4.6 in blind testing. Uses Reinforcement Learning Environments (RLEs) – custom "training gyms" for enterprise-specific agents.

### Overall Thesis

Thompson argues that Microsoft's Build keynote showed a company pivoting away from Windows-centric local AI toward a cloud-as-hub model (Project Solara, enterprise MAI models). The Nvidia RTX Spark represents the old thinking (local inference for chatbots), while Project Solara and MAI models represent the future.
